\doxysubsection{Detailed Description}
Header file containing functions for printing formatted output to the console. 

This header file defines functions for printing various types of formatted output to the console, including loading animations, titles, options, directories, helper messages, information in a monospaced font, lists of code values, tables with double values, regular tables, and lists of compared values. It also includes color schemes and utility functions for centering text and getting terminal width.
